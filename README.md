# Web Thông Tin Cá Nhân

Một website giới thiệu cá nhân xây dựng bằng Vue 3 và Vite.

## Chạy dự án

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## VNUA API relay (khi server bi chan IP)

Neu server dat o Singapore bi chan boi he thong dao tao, tao file `.env` va dat endpoint relay/proxy tai VN:

```bash
VITE_VNUA_API_BASE=https://your-vn-relay.example.com
```

Mac dinh app se goi `/vnua-api` (proxy local).
