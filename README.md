# Web Thông Tin Cá Nhân

Một website giới thiệu cá nhân xây dựng bằng Vue 3 và Vite.

## Chạy dự án

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Nội dung cần thay thế

- `Tên của bạn`
- `Front-end / Vue Developer`
- Số điện thoại, email và các mô tả mẫu trong trang
